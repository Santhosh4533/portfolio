# Cinematic Portfolio | Landon Norris Inspired

A high-end, premium portfolio website built with modern technologies.

## Features
- **Cinematic Design**: Fullscreen hero, large typography, and premium contrast.
- **Smooth Interaction**: Lenis smooth scroll and custom cursor logic.
- **Premium Animations**: Framer Motion scroll-reveal, exit-animations, and hover-glow effects.
- **Responsive & Dynamic**: Fully mobile-optimized with an interactive menu.
- **Backend-Ready**: Integrated Express API for contact submissions.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Framer Motion, Lenis, Lucide Icons.
- **Backend**: Node.js (Express), CORS, Dotenv.
- **Hosting**: Vercel (Frontend), Render/Railway (Backend).

## Getting Started

### 1. Start the Backend
```bash
cd server
npm install
npm start
```

### 2. Start the Frontend
```bash
cd client
npm install
npm run dev # Runs on http://localhost:3000
```

### 3. Build for Production
```bash
cd client
npm run build
```

## Folder Structure
- `/client/src/components`: Reusable UI components (Hero, About, Projects, etc.).
- `/client/src/hooks`: Custom React hooks (useMousePosition).
- `/client/src/animations`: Framer Motion presets.
- `/server/index.js`: Main API handler.
